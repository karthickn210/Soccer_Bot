# Soccer_Bot
Mobile Controlled Soccer Robot

### Components Needed

1. arduino
2. HC-05 bluetooth module
3. L298N motor driver module
4. 2 X 12V DC motor
5. Servo motor
6. 12V battery 
7. Chassis
8. Jumper wires
9. buzzer
10. Tyres
